# Smart Plant Watering System

IoT project using ESP8266 and Blynk to monitor soil moisture and control a water pump.  
Features real-time moisture monitoring, relay-based pump control, and LCD display for status.  

**Hardware:** ESP8266, soil moisture sensor, relay module, water pump, LCD 16x2 I2C  
**Software:** Arduino IDE, Blynk, LiquidCrystal_I2C  

Full code is in the .ino file.
